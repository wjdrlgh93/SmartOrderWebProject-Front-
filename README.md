## 🐕‍🦺 EVERYTHING ABOUT PET
> EVERYTHING ABOUT PET 은 최근 증대하는 반려동물 시장의 니즈에 맞추어 \
> 무인숍의 위치부터 결제 & 회원관리까지 한번에 \
> 처리가능한 스마트오더 시스템 반려동물 상품 판매 플랫폼입니다 



## 💝기획의도 및 활용방안

📈 기획의도 <br>
 - 펫용품은 반복 구매가 많아 정기 배송이나 간편한 주문이 가능한 온라인 쇼핑몰의 장점이 잘 드러남.
 - 반려동물을 키우는 가구가 지속적으로 증가하고 있으며, 펫용품에 대한 수요도 함께 늘어나고 있기때문에 펫샵으로 결정.
 - CURD, 회원관리, 장바구니, 카테고리, 정기배송 후기등록 등.. 웹 개발의 전형적인 전 과정을 경험할 수 있어 학습 측면에서 효과적이라 판단.
<br>

📢 활용방안 <br>

- 24시간 비대면 쇼핑플랫폼
  > 반려동물 용품,사료,간식, 의류 위생용품등 다양한 상품을 시간제약없이 구매가능.<br>

- O2O 연계 서비스 제공
  > 오프라인 매장 및 병원과 연계한  예약 시스템 운영 가능 

- 반려동물 전문 정보 제공
  > 사료 간식 종류별 비교, 질병예방정보, 관리 팁 등 콘텐츠 제공이 가능함 

- 정기배송 서비스
  > 다양한 펫샵 용품 반복 무개 상품은 정기배송으로 자동화 하여, 재고 부족 걱정 해소 + 고객 충성도 향상을 노릴 수 있음.

## 🛠 주요기능

| 주요기능 | 설명 |
|------|------|
| 🔑인증/인가 | 가상 토큰방식으로 JSON 가상 DB 를 이용한 로그인/회원가입 구현 |
| 🛒상품 | 상품목록 비동기 로딩, 상품 카테고리/검색 필터링 가능 |
| 🛠상태관리 | ⚛ Redux 를 활용한 전역상태 관리 및 장바구니 상태 동기화 |
| 💰결제/주문 | ⚛ Slice 를 활용한 가상결제 기능 구현 |
| 🎨UI/UX | Media Query 를 사용한 반응형 웹 구현, 사용자 경험을 위한 커스텀 Hook 활용 및 헤더/푸터 네이게이션 바 구현 |
